Projects

StudyMate

StudyMate is an online education app for schools/colleges to virtually operate classes, it also has video conferencing feature for live classes. This app connects people who want to learn and who want to teach.

Technology

StudyMate is built using Java with Material Design Library for Clean and Modern UI. Cloud Firestore provides the Database Solution and Retrofit handles the HTTP traffic. Jitsi Meet SDK is implemented to provide Video Conferencing Functionality and Glide for caching and loading the user profile pictures. User Authentication management is handled using Firebase Authentication service. Cloud Functions framework is used to handle backend code in response to events triggered by Firebase features and HTTP requests.
